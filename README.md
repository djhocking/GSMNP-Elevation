## Run models

In the `Code/` directory.

1.  `prep_data.R`
2.  `set_conditions.R`
3.  `Stan_Models/final_od.R` - `Stan_Models/final_od.stan` (Optional:
    other models for model comparison)
4.  `check_sims.R`
5.  `compare_models.R` (Optional)
6.  `summarize.R` (under development)
