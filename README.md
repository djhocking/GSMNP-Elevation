## Run models

1.  `prep_data.R`
2.  `final_od.R` or `site_od_full_varK.R` (Opt: other models for model
    comparison)
3.  `check_sims.R` (coming soon)
4.  `compare_models.R` (optional)
5.  `summarize.R` (coming soon)
